## Backend
2 terminal sessions needed for development!
- #1 session - compile TypeScipt to JavaScript after changes
```bash
npm run watch
```
- #2 session - restart server after file changes
```bash
npm run start-dev
```

run production server
```bash
npm run start-prod
```

run tests
```bash
npm test
```
